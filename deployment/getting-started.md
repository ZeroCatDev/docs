# 开始使用

ZeroCat是一个轻量级的在线编程、分享平台。本文档将引导您完成部署过程。

## 系统要求

- Node.js 14+
- npm 或 yarn
- 数据库（MySQL/PostgreSQL）
- 推荐使用Docker进行部署

## 快速开始

如果您想快速启动项目，可以使用以下命令：

```bash
# 克隆仓库
git clone https://github.com/ZeroCatDev/ZeroCat.git
cd ZeroCat

# 安装依赖
npm install

# 启动项目
npm start
```

更多详细配置，请参考[基础配置](./basic-config.md)文档。
